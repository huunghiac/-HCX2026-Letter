# Thư mục chứa nhạc nền (Audio)

Bạn vui lòng đặt file nhạc **Until I Found You (Violin Cover)** định dạng `.mp3` vào thư mục này.

### Hướng dẫn:
1. Chuẩn bị file nhạc `.mp3` của bài hát.
2. Đổi tên file thành: `until_i_found_you_violin_cover.mp3`
3. Di chuyển/sao chép file đó vào thư mục `audio/` này.

Đường dẫn mặc định trong file `data.json` đang được thiết lập là `audio/until_i_found_you_violin_cover.mp3`.
